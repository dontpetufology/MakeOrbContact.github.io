# assets/

Place media and derived files here:

- /assets/media/… — put thumbnails, video files, glyph images, audio, PDFs
- /assets/css/styles.css — site CSS
- /assets/js/search.js — client search script

Example structure:
- assets/media/thumbnails/mantra-07.jpg
- assets/media/videos/ORB-20260101-07.mp4
- assets/media/glyphs/G-101.png
